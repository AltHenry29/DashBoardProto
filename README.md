Se utilizo JavaFX, .css, y se cargaron los elementos/metodos en cada uno de los elementos java para su ejecucion.

Esta estructurado asi:

src/
|
├── main.java.com.example.laterals/
│   └── DashboardController.java
│   └── HelloAplication.java
│   └── HelloController.java
│   └── Producto.java
│
└── main.resources.com.example.laterals/
    └── dashboard.fxml
    └── dashboard.fxml
    └── Styledb.css
    └── styles.css

Las instrucciones de ejecucion son estas:
1. En HelloAplication.java ejecutelo como si fuera un main
2. Se abrira la ventana con el DashBoard, puede realizar funciones de prueba para guardar, eliminar, actualizar y limpiar los productos
3. De click en cerrar sesion y lo llevara al login
4. Para volver al DashBoard puede darle las credenciales admin tanto para usuario como contraseña para el rol de administrador o cajero
   en todos los campos para ingresar e interactuar de nuevo con el dashboard.
